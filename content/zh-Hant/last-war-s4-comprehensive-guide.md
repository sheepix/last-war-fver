本文是《Last War: Survival》第四賽季（S4 - 永夜之島）的完整攻略，涵蓋新機制解讀、英雄推薦、陣容搭配、建築升級優先順序、每週攻略要點以及賽季過渡建議。

## 一、S4 核心新機制概覽

### 🌙 賽季主題：永夜之島

第四賽季引入了**黑夜機制**，整個地圖被黑暗籠罩，需要依靠**燈塔**來驅散黑暗。

### 🔑 新增核心系統

| 系統 | 說明 |
|------|------|
| **燈塔** | 驅散黑暗，提供光明增益和病毒抗性 |
| **發電機** | 為燈塔供電（最多4個個人發電機+1個聯盟發電機） |
| **電工** | 新助手，可為部隊提供照明，解除血夜殭屍 Buff |
| **光電實驗室** | 提升病毒抗性，解鎖發電機和電工 |
| **石英工坊** | 生產石英（最多4個，週票+1額外） |
| **戰術卡牌** | 新增戰鬥系統，2張核心卡+8張額外卡 |
| **勾玉商店** | 用勾玉兌換石英、戰術卡包等 |

### ⚡ 電力管理系統

S4 的核心玩法圍繞**電力管理**展開：

- **資源鏈**：石頭 → 石英工坊 → 石英 → 光電實驗室 → 病毒抗性 → 戰鬥效能
- **血夜期間**：每分鐘消耗 18,000 電力，大多數玩家儲備會耗盡
- **提示**：務必開啟**自動調整設定**，防止電力耗盡後燈光熄滅

## 三、S4 建築升級優先順序

### 🏗️ Day 1 立即執行

**升級順序（非常重要）**：

1. **擊殺 Lv.190 末日行者** → 獲得首殺獎勵 207K 石頭
2. **擊殺 Lv.1-3 小殭屍** → 取得首殺獎勵（無需抗性）
3. **佔領挖掘據點** → 每小時取得石頭

### 📋 建築優先順序清單

| 優先順序 | 建築 | 目標等級 | 說明 |
|--------|------|----------|------|
| 🥇 | 光電實驗室 | 35級 | 解鎖更多發電機和電工，賽季最高優先順序 |
| 🥈 | 燈塔 | 10級→35級 | 只用鐵和食物，10級後需等光電35級才能繼續 |
| 🥉 | 石英工坊 | 15級+ | 從15級開始建造，每級成本增加但產出相同 |
| ⭐ | 聯盟中心 | 隨HQ | 協助/增援容量 |
| ⭐ | 醫院 | 12級+ | 保存70%+部隊 |

### ⚠️ 重要提示

- 石英工坊從15級開始建造，因為每級成本增加但產出相同
- 光電實驗室達到35級後才能繼續升級燈塔
- 為第一支部隊配備**電工**，解除血夜殭屍 Buff

## 四、血夜活動攻略

### 🌙 血夜時間表（每天3次，每次30分鐘）

| 時間 | 說明 |
|------|------|
| 02:30 | 伺服器時間 |
| 10:30 | 伺服器時間 |
| 18:30 | 伺服器時間 |

### 血夜規則

- 殭屍獲得 **50% Buff**，變得更強
- 燈塔電力需求急劇增加
- **電工部隊**可點亮道路，解除殭屍 Buff
- 黑暗中基地可能遭到遊蕩殭屍攻擊
- 擊敗殭屍有更高機率獲得戰術卡包
- 取得**勾玉**（個人+聯盟目標）

### 🎯 勾玉商店兌換優先順序

每週刷新，務必兌換：

1. ✅ **石英**（最高優先順序）
2. ✅ 戰術 SR/UR 卡包
3. ✅ 霸主訓練證書和補給箱
4. ✅ 招財貓裝飾（第2週起）

## 五、戰術卡牌系統

### 🃏 卡牌配置

- **2張核心卡**（含特殊戰術能力，UR非常稀有）
- **8張額外卡**（4戰鬥+4資源）

### 取得方式

- 擊敗血夜殭屍
- 勾玉商店購買
- 每天最多取得20張新卡

### 💡 0 課卡牌策略

- 保留每種卡牌類型，直到需要或取得配套卡
- 分解重複卡牌，用碎片升級其他卡
- 優先升級核心卡

### 推薦卡組（通用配置）

**主配置**：
- 風馳者（Windrusher）
- 急速行軍（Rapid March）
- 傷害減免逆轉 或 傷害逆轉
- 混編陣容（Mixed Formation）
- 4+1 戰術

## 七、推薦陣容詳解

### 陣容一：4 坦克 + 1 亞當混編（平民/微課推薦）

利用賽季卡牌實現跨陣營加成。

**陣容構成**：
- 前排：**史嘉蕾** + **金伯利**
- 後排：**亞當** + **馬歇爾** + **墨菲**

**核心機制**：
- 亞當為前排提供「防守反擊」狀態，受擊時觸發普攻反擊
- 史嘉蕾普攻倍率極高（可達 1400% 物理傷害），配合反擊輸出爆炸
- 馬歇爾鎖定敵方最高攻單位，協助集火

**優勢**：
- 克制飛機隊（史嘉蕾打物理傷害，無視能量減傷）
- 同戰力強度高，可跨戰力壓制
- 對平民友好，英雄取得相對容易

---

### 陣容二：傳統飛機隊（Air Squad）

**核心英雄**：DVA + 盧修斯 + 卡莉 + 斯凱勒 + 莫里森

**優勢**：
- 高能量減傷，對抗技能傷害有優勢
- 爆發能力強，快速擊殺脆皮

**劣勢**：
- 被物理輸出陣容（如史嘉蕾坦克隊）克制

---

### 陣容三：坦克主力陣容（S4 推薦）

S4 坦克陣容獲得 25% 同類型部隊加成，非常強勢。

**陣容**：威廉姆斯 + 墨菲 + 金伯利 + 特斯拉 + 斯特曼

**說明**：3-4 坦克構成最優，配合賽季機制非常強。

## 八、S4 每週攻略要點

### 📅 Week 1：基礎建設

**解鎖內容**：
- 光電實驗室、石英工坊、聯盟中心
- 戰術卡牌、燈塔
- 血夜降臨（準備階段）
- 清理作戰（首殺獎勵）

**重點任務**：
- 擊殺 Lv.190 末日行者取得石頭
- 佔領挖掘據點
- 將燈塔升至10級
- 擊殺 Lv.1-3 小殭屍取得首殺獎勵

### 📅 Week 2：血夜獵人

**解鎖內容**：
- 血夜獵人 PvP 活動
- 美食聖殿
- 神樹祝福
- 惡魔巡邏車

**血夜獵人規則**：
- 需要血夜獵人職業技能（約70級）
- 獵人獲得：+250病毒抗性、+20%士氣、+10%行軍速度
- 獵人之間可以互相攻擊
- 自動進入戰爭狂熱狀態（無法開盾）

### 📅 Week 3：英雄升級與派系選擇

**解鎖內容**：
- 莎拉 UR 升級
- 貿易戰
- **派系選擇**（影之里 vs 甲部隊）

**派系選擇提示**：
- 兩個派系：影之里 和 甲部隊
- 每個派系4個戰區
- 影響後續銅戰爭宣戰順序
- 務必與聯盟協調選擇

### 📅 Week 4+：銅戰爭

**核心目標**：收集盡可能多的**銅**

- 透過佔領城市每小時取得銅
- 用於升級聯盟中心
- 派系決鬥決定最終賽季排名
- 共8輪（每週2輪）

**銅戰爭技能**：
- **進攻方**：戰爭領主的原子吐息
- **防守方**：管家的特斯拉線圈

## 九、S3 過渡 S4 建議

### 過渡前準備（S3 末期）

1. **囤積體力** — 需要大量體力擊殺世界殭屍和參與賽季活動
2. **儲備雷達任務** — 賽季開始時快速完成
3. **停止擊殺末日行者** — 賽季前1-2天停止，等S4首殺獎勵
4. **保存英雄回程票** — S4早期使用
5. **購買紫色碎片** — 為莎拉 UR 升級準備
6. **購買當前賽季基地外觀** — 在 S4 使用其 Buff

### S4 開啟後（Day 1-7）

1. **不要立即擊殺末日行者** — 等賽季開始後再擊殺取得首殺獎勵
2. **優先將光電實驗室升至 35 級**
3. **解鎖石英工坊** — 開始石頭→石英鏈條
4. **開啟血夜自動調整設定**
5. **擊殺 Lv.1-3 小殭屍** — 無需抗性即可取得首殺獎勵
6. **盡快佔領挖掘據點** — 為所有聯盟成員提供每小時資源
7. **每天擊殺 190 級末日行者**

## 十、0 課玩家日常最佳化

### ⏰ 每日必做

| 時間 | 行動 |
|------|------|
| 每4小時 | 登入收取體力、資源 |
| 血夜時間 | 參與血夜活動，擊敗殭屍取得勾玉 |
| 日常 | 完成所有日常任務 |
| 聯盟 | 參與所有聯盟活動 |

### 💡 效率技巧

**聯盟協調**：
- 低抗性盟友發起集結，高抗性盟友加入完成
- 互相派遣電工提升光明狀態
- 每日派遣電工為神樹充電

**資源管理**：
- 所有賽季建築優先使用聯盟科技「老工匠」（降低建造成本）
- 工程師職業開局（降低建造和研究成本）
- 批量訓練部隊，利用加成活動

**活動參與**：
- 只參與免費層級和可達成的目標
- 囤積代幣，等待獎勵加成活動
- 優先完成聯盟活動取得團隊獎勵
