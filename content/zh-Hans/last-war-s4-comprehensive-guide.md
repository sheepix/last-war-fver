# Last War S4 赛季完全攻略：永夜之岛机制、阵容与进阶指南

> 最后修改：2025-03-21
> 适用赛季：第四赛季 - 永夜之岛（Evernight Isle）

本文是《Last War: Survival》第四赛季（S4 - 永夜之岛）的完整攻略，涵盖新机制解读、英雄推荐、阵容搭配、建筑升级优先级、每周攻略要点以及赛季过渡建议。

## 一、S4 核心新机制概览

### 🌙 赛季主题：永夜之岛

第四赛季引入了**黑夜机制**，整个地图被黑暗笼罩，需要依靠**灯塔**来驱散黑暗。

### 🔑 新增核心系统

| 系统 | 说明 |
|------|------|
| **灯塔** | 驱散黑暗，提供光明增益和病毒抗性 |
| **发电机** | 为灯塔供电（最多4个个人发电机+1个联盟发电机） |
| **电工** | 新助手，可为部队提供照明，解除血夜僵尸 Buff |
| **光电实验室** | 提升病毒抗性，解锁发电机和电工 |
| **石英工坊** | 生产石英（最多4个，周票+1额外） |
| **战术卡牌** | 新增战斗系统，2张核心卡+8张额外卡 |
| **勾玉商店** | 用勾玉兑换石英、战术卡包等 |

### ⚡ 电力管理系统

S4 的核心玩法围绕**电力管理**展开：

- **资源链**：石头 → 石英工坊 → 石英 → 光电实验室 → 病毒抗性 → 战斗效能
- **血夜期间**：每分钟消耗 18,000 电力，大多数玩家储备会耗尽
- **提示**：务必开启**自动调整设置**，防止电力耗尽后灯光熄灭

## 三、S4 建筑升级优先级

### 🏗️ Day 1 立即执行

**升级顺序（非常重要）**：

1. **击杀 Lv.190 末日行者** → 获得首杀奖励 207K 石头
2. **击杀 Lv.1-3 小僵尸** → 获取首杀奖励（无需抗性）
3. **占领挖掘据点** → 每小时获得石头

### 📋 建筑优先级清单

| 优先级 | 建筑 | 目标等级 | 说明 |
|--------|------|----------|------|
| 🥇 | 光电实验室 | 35级 | 解锁更多发电机和电工，赛季最高优先级 |
| 🥈 | 灯塔 | 10级→35级 | 只用铁和食物，10级后需等光电35级才能继续 |
| 🥉 | 石英工坊 | 15级+ | 从15级开始建造，每级成本增加但产出相同 |
| ⭐ | 联盟中心 | 随HQ | 帮助/增援容量 |
| ⭐ | 医院 | 12级+ | 保存70%+部队 |

### ⚠️ 重要提示

- 石英工坊从15级开始建造，因为每级成本增加但产出相同
- 光电实验室达到35级后才能继续升级灯塔
- 为第一支部队配备**电工**，解除血夜僵尸 Buff

## 四、血夜活动攻略

### 🌙 血夜时间表（每天3次，每次30分钟）

| 时间 | 说明 |
|------|------|
| 02:30 | 服务器时间 |
| 10:30 | 服务器时间 |
| 18:30 | 服务器时间 |

### 血夜规则

- 僵尸获得 **50% Buff**，变得更强
- 灯塔电力需求急剧增加
- **电工部队**可点亮道路，解除僵尸 Buff
- 黑暗中基地可能遭到游荡僵尸攻击
- 击败僵尸有更高几率获得战术卡包
- 获得**勾玉**（个人+联盟目标）

### 🎯 勾玉商店兑换优先级

每周刷新，务必兑换：

1. ✅ **石英**（最高优先级）
2. ✅ 战术 SR/UR 卡包
3. ✅ 霸主训练证书和补给箱
4. ✅ 招财猫装饰（第2周起）

## 五、战术卡牌系统

### 🃏 卡牌配置

- **2张核心卡**（含特殊战术能力，UR非常稀有）
- **8张额外卡**（4战斗+4资源）

### 获取方式

- 击败血夜僵尸
- 勾玉商店购买
- 每天最多获得20张新卡

### 💡 0 氪卡牌策略

- 保留每种卡牌类型，直到需要或获得配套卡
- 分解重复卡牌，用碎片升级其他卡
- 优先升级核心卡

### 推荐卡组（通用配置）

**主配置**：
- 风驰者（Windrusher）
- 急速行军（Rapid March）
- 伤害减免逆转 或 伤害逆转
- 混编阵容（Mixed Formation）
- 4+1 战术

## 七、推荐阵容详解

### 阵容一：4 坦克 + 1 亚当混编（平民/微氪推荐）

利用赛季卡牌实现跨阵营加成。

**阵容构成**：
- 前排：**斯嘉丽** + **金伯利**
- 后排：**亚当** + **马歇尔** + **墨菲**

**核心机制**：
- 亚当为前排提供"防守反击"状态，受击时触发普攻反击
- 斯嘉丽普攻倍率极高（可达 1400% 物理伤害），配合反击输出爆炸
- 马歇尔锁定敌方最高攻单位，协助集火

**优势**：
- 克制飞机队（斯嘉丽打物理伤害，无视能量减伤）
- 同战力强度高，可跨战力压制
- 对平民友好，英雄获取相对容易

---

### 阵容二：传统飞机队（Air Squad）

**核心英雄**：DVA + 卢修斯 + 卡莉 + 斯凯勒 + 莫里森

**优势**：
- 高能量减伤，对抗技能伤害有优势
- 爆发能力强，快速击杀脆皮

**劣势**：
- 被物理输出阵容（如斯嘉丽坦克队）克制

---

### 阵容三：坦克主力阵容（S4 推荐）

S4 坦克阵容获得 25% 同类型部队加成，非常强势。

**阵容**：威廉姆斯 + 墨菲 + 金伯利 + 特斯拉 + 斯特曼

**说明**：3-4 坦克构成最优，配合赛季机制非常强。

## 八、S4 每周攻略要点

### 📅 Week 1：基础建设

**解锁内容**：
- 光电实验室、石英工坊、联盟中心
- 战术卡牌、灯塔
- 血夜降临（准备阶段）
- 清理作战（首杀奖励）

**重点任务**：
- 击杀 Lv.190 末日行者获取石头
- 占领挖掘据点
- 将灯塔升至10级
- 击杀 Lv.1-3 小僵尸获取首杀奖励

### 📅 Week 2：血夜猎人

**解锁内容**：
- 血夜猎人 PvP 活动
- 美食圣殿
- 神树祝福
- 恶魔巡逻车

**血夜猎人规则**：
- 需要血夜猎人职业技能（约70级）
- 猎人获得：+250病毒抗性、+20%士气、+10%行军速度
- 猎人之间可以互相攻击
- 自动进入战争狂热状态（无法开盾）

### 📅 Week 3：英雄升级与派系选择

**解锁内容**：
- 莎拉 UR 升级
- 贸易战
- **派系选择**（影之里 vs 甲部队）

**派系选择提示**：
- 两个派系：影之里 和 甲部队
- 每个派系4个战区
- 影响后续铜战争宣战顺序
- 务必与联盟协调选择

### 📅 Week 4+：铜战争

**核心目标**：收集尽可能多的**铜**

- 通过占领城市每小时获得铜
- 用于升级联盟中心
- 派系决斗决定最终赛季排名
- 共8轮（每周2轮）

**铜战争技能**：
- **进攻方**：战争领主的原子吐息
- **防守方**：管家的特斯拉线圈

## 九、S3 过渡 S4 建议

### 过渡前准备（S3 末期）

1. **囤积体力** — 需要大量体力击杀世界僵尸和参与赛季活动
2. **储备雷达任务** — 赛季开始时快速完成
3. **停止击杀末日行者** — 赛季前1-2天停止，等S4首杀奖励
4. **保存英雄回程票** — S4早期使用
5. **购买紫色碎片** — 为莎拉 UR 升级准备
6. **购买当前赛季基地皮肤** — 在 S4 使用其 Buff

### S4 开启后（Day 1-7）

1. **不要立即击杀末日行者** — 等赛季开始后再击杀获取首杀奖励
2. **优先将光电实验室升至 35 级**
3. **解锁石英工坊** — 开始石头→石英链条
4. **开启血夜自动调整设置**
5. **击杀 Lv.1-3 小僵尸** — 无需抗性即可获取首杀奖励
6. **尽快占领挖掘据点** — 为所有联盟成员提供每小时资源
7. **每天击杀 190 级末日行者**

## 十、0 氪玩家日常优化

### ⏰ 每日必做

| 时间 | 行动 |
|------|------|
| 每4小时 | 登录收取体力、资源 |
| 血夜时间 | 参与血夜活动，击败僵尸获取勾玉 |
| 日常 | 完成所有日常任务 |
| 联盟 | 参与所有联盟活动 |

### 💡 效率技巧

**联盟协调**：
- 低抗性盟友发起集结，高抗性盟友加入完成
- 互相派遣电工提升光明状态
- 每日派遣电工为神树充电

**资源管理**：
- 所有赛季建筑优先使用联盟科技"老兵工匠"（降低建造成本）
- 工程师职业开局（降低建造和研究成本）
- 批量训练部队，利用加成活动

**活动参与**：
- 只参与免费层级和可达成的目标
- 囤积代币，等待奖励加成活动
- 优先完成联盟活动获取团队奖励

## 十一、常见错误避免

| ❌ 错误 | ✅ 正确做法 |
|---------|------------|
| 英雄升级过于分散 | 专注核心三人组 |
| 忽视仓库升级 | 优先升级仓库防止被掠夺 |
| 忘记部队晋升 | 及时进行部队晋升 |
| 不侦查就攻击 | 侦查后再决定攻击 |
| 留在不活跃联盟 | 加入活跃联盟 |
| 随机花费资源 | 规划资源使用，配合活动 |
| 赛季结束不拆解卡牌 | 赛季结束前拆解重复卡牌 |
| 忽视派系选择 | 与联盟协调选择 |

## 十二、赛季目标总结

### 🎯 0 氪玩家赛季目标

1. ✅ 光电实验室达到35级（最高优先级）
2. ✅ 灯塔升至10级（前期），35级后继续升级
3. ✅ 石英工坊全部15级+
4. ✅ 莎拉完成 UR 升级
5. ✅ 积极参与血夜活动，积累勾玉兑换资源
6. ✅ 收集战术卡牌，构建适合自己的卡组
7. ✅ 与联盟密切配合，共享电工、协调攻击

### 🏆 成功关键

- **耐心即力量** — 长期规划胜过即时满足
- **效率至上** — 有限资源最大化产出
- **团队依赖** — 强大的联盟关系至关重要
- **战略时机** — 资源使用的最佳时机
- **淡季优势** — 利用休赛期做准备
